# Team Notes Glossary

Small shared vocabulary for the Team Notes example.

## Terms

- User: a registered person who can sign in.
- Session: authenticated proof that a user can perform note actions.
- Note: shared team content that starts active and may become archived.
- Archive: remove a note from the active list without deleting its record.

## Usage

Use these terms in UI copy, API tests, and Forge docs so the example stays
consistent across product, runtime, and data views.
